# GameHub

## Set Up

1. Install Node LTS on your computer, if you haven't already.
2. Open your terminal and type `git clone https://github.com/IT3049C/final-project-kat-austin.git` to copy the repo to your machine.
3. Navigate to the project directory with `cd final-project-kat-austin`.
4. Run `npm install` to install project dependencies.

## Running the Project

To run the project in development mode, use the command `npm run dev`. To see the project in production mode, run `npm run build`, then `npm run preview`.

## Testing

Testing is conducted in this project with Playwright. To run all Playwright tests, use the command `npm test`.
